# SOP — Disposition MRO Product
**Process ID:** SCOR-SR2.2
**Framework:** SCOR | **Domain:** Return
**Generated:** 2026-06-07

## Purpose
Process of determining appropriate disposition for MRO items including return to supplier, internal reuse, donation or disposal

## Triggers
- MRO Item return identification
- Item Condition Assessment completion

## Inputs Required
- MRO return identification
- item condition assessment
- supplier return policy
- reuse opportunities

## Process Steps
1. IF MRO Item condition is reusable THEN consider Reuse Opportunities
2. IF Supplier Return Policy allows return THEN consider Return to Supplier
3. IF MRO Item is hazardous THEN follow Hazardous Materials Regulations

## Expected Outputs
- disposition decision
- return or reuse plan
- disposal instructions

## Business Rules
- rule1: MRO Item must be evaluated for reuse before disposal
- rule2: Disposition Decision must comply with Environmental Compliance regulations
- rule3: Disposal Instructions must follow Asset Disposal Policy

## Exception Handling
- exception1: IF MRO Item is missing or damaged THEN notify Supplier and update Disposition Decision
- exception2: IF Reuse Opportunities are not available THEN consider Donation or Disposal

## Success Criteria
- Disposition Decision is made within Disposition Cycle Time
- MRO Recovery Value is optimized
- Reuse Rate is above target

## Compliance Requirements
- environmental compliance
- hazardous materials regulations
- asset disposal policy
